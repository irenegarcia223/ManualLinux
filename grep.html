Grep

Se trata de un seleccionador de filas. Nos permite buscar palabras o cadenas de texto en archivos o en la entrada estándar de la terminal. Cuando localiza lo que se está buscando muestra las líneas completas que contienen esa información específica.

Sus opciones más importantes son:

  • -i: la búsqueda no distinguirá entre mayúsculas y minúsculas. Ejemplo: grep -i "mensaje" archivo.txt
  • -c: solo mostrará el número de líneas que coinciden con el patrón buscado. Ejemplo: grep -c "mensaje" archivo.txt
  • -E: Esta opción permite usar expresiones regulares para buscar patrones más complejos. Ejemplo: grep -E "[0-9]{3}-[0-9]{3}-[0-9]{4}" archivo.txt
  • -n: Esta opción muestra el número de línea de cada línea que coincide con el patrón de búsqueda. Ejemplo: grep -n "hola" archivo.txt
